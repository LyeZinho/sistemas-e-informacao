Sistemas

Um sistema é um conjunto de organismos
que trabalham em conjunto para um objetivo.
























Um exemplo é o sistema digestivo
que trabalham em conjunto com varios orgãos
para realizar a digestão de alimentos no corpo
humano.
























O sistema nervoso pode ser considerado
um subsistema, pois trabalha em conjunto com
outros orgãos dentro de um sistema maior que é o corpo
humando.



































Subsistemas

Um subsistema e quando um sistema menor
opera dentro de um sistema maior.























Dados

Um dado e uma representação de um objeto, fato, pessoa
ou outros tipos de entidades ou acontecimentos podem essas
representações podem ser símbolos, números,
palavras ou outros tipos de sinais.








Informação

Um dado e uma representação de um objeto, fato, pessoa
ou outros tipos de entidades ou acontecimentos podem essas
representações podem ser símbolos, números,
palavras ou outros tipos de sinais.












Uma informação atual e correta só é possível se os seus dados estiverem atualizados e forem precisos. De outra forma, a nossa informação não será útil.





Mas, para que isto possa acontecer, existem algumas condições que os dados devem garantir.

  • Atualidade
  • Correção
  • Relevância
  • Disponibilidade
  • Legibilidade




Para as organizações e os seus sistemas de informação, os dados são vistos como registros em bruto, ou seja, antes de serem tratados e organizados. Por sua vez, a informação é constituída por dados depois de tratados e organizados de forma significativa para os seres humanos.

Para uma empresa ou outro tipo de organização, a informação refere-se a objetos ou acontecimentos com importância para o funcionamento dessa organização, tais como: mercadorias, compras, vendas, pessoas, processos de trabalho, etc.










Sistemas de Informação

Conjunto de recursos utilizados para recolher e
tratar os dados e a informação que são importantes para
a atividade de uma organização.
